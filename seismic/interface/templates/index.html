{% extends "base.html" %} {% block title %}Home{% endblock %} {% block heading %}Seismic Detection with SAX{% endblock %} {% block explain %}Instructions as listed in Appendix A.2.2 in the report{% endblock %} {% block contents %}
  1. Select Import from the menu. Then click on Choose Files and select some observations (there are some under /sample/_all in the included source or at https://github.com/t0mmyt/seismic-detector/tree/master/sample/_all). Then hit Upload, if all was successful, you will see a list of the files uploaded in green.
  2. Once the files have been uploaded, click on Observations.
  3. Then select a Network, Station and Channel to see observations that match your criteria.
  4. By ticking the checkbox next to an event, you are presented with a rendering of the whole observation and some controls
  5. To detect an event either by STA/LTA or SAX, hit the relevant control and then hit the light blue Refresh button next to them to see if there were any potential events found. If there were, you can tick the box next the the number of events to see them displayed on the graph.
  6. To see the event in more detail and to view the PAA/SAX process, you can click on the event ID for that event.
